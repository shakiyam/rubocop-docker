rubocop-docker
==============

[RuboCop](https://github.com/rubocop-hq/rubocop) Docker Image

How to Use
----------

```console
docker container run --rm -t -u "$(id -u):$(id -g)" -v "$PWD":/work:ro docker.io/shakiyam/rubocop
```
